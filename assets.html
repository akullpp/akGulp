Namespace: assets

assets

Copies unhandled assets with preserved paths to dist.

Unhandled assets can be configered via paths.handledAssets. The unhandled assets are all files with file extensions in paths.app.root except the files listed in the aforementioned path array.

The files should be handled as soon as possible.

config.paths.app.root/**/* -> config.paths.build.dist.base/config.paths.app.root
