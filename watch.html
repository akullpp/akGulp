Namespace: watch

watch

Watches styles, scripts, tests, the main HTML and bower file for changes.

Respectively, see:

  • styles for paths.app.styles
  • watch:scripts for paths.app.scripts
  • watch:tests for paths.tests

If there is a change in the paths.app.mainHtml detected the deps task is executed.

Used internally by dev.
